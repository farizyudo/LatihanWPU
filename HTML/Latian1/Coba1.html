HTML Dasar : Image

img : tag html untuk menaruh gambar
Scr : Atribut untuk Menaruh link gambar
Alt : Atribut Untuk memberi penjelasan semisal gambar tidak muncul
Title : Atribut Untuk memberi Nama pada gambar

Height : Atribut Untuk Menetapkan panjang gambar (dalam px : Pixel, dan % : Relatife terhadap halaman)

HTML Dasar : Table

Truktur tabel

Baris / Row : Horizontal

Kolom / Column : Vertikal

Cell / data : Perpotongan Baris dan kolom

Table : Tag pembuat tabel
tr(Table Row) : Tag baris
td(Table Data) : Tag Kolom
thead (Table Head) : Tag Kepala Judul
th(Table Headeng) : Tag Judul untuk menyimpan data
tbody (Table Body) : Tag Body

Atribut :
ini pake css (kudune)
border : Menambahkan border / garis disekitar cell
cellspacing : Menambah jarak antar cell
cellpadding : Memberi luas pada sebuah cell

Ini mah dimateri selanjutnya :

rowspan
